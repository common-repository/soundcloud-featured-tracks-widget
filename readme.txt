Stable tag: trunk

Creates a widget that inserts up tp 4 featured tracks with admin panel management.

== Description ==

You can display:

* Soundcloud tracks
* Any widgetized areas
* Audio



Version 1.0
First version of the plugin.  

== Installation ==

1. Upload `soundcloud-widget.php`  to the `/wp-content/plugins/` directory.
2. Activate the plugin via the Plugins menu.
3. Visit the feature tracks Admin Panel and paste the embed code of the tracks
4. Add widget to a sidebar area or any other widgetized area.
5. Ability to add to multiple areas.
